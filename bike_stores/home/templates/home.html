Cấu Trúc Dự Án

App: production

  • • Category - Quản lý danh mục sản phẩm
  • • Brand - Quản lý thương hiệu
  • • Product - Quản lý sản phẩm
  • • Stock - Quản lý kho hàng

App: sales

  • • Customer - Quản lý khách hàng
  • • Store - Quản lý cửa hàng
  • • Staff - Quản lý nhân viên
  • • Order & OrderItem - Quản lý đơn hàng

Cấu Trúc Cơ Sở Dữ Liệu

PRODUCTION SCHEMA
Categories → Brands → Products
Products ↔ Stocks ↔ Stores
SALES SCHEMA
Customers → Orders ← Staff
Orders → OrderItems → Products
Staff → Stores (Manager hierarchy)
KEY RELATIONSHIPS
• Foreign Keys: brand_id, category_id, store_id
• Composite Keys: (store_id, product_id)
• Self-referencing: manager_id → staff_id

Hướng Dẫn Cài Đặt

1. Clone Repository

git clone https://github.com/lamlv723/python-uit
cd python-uit

2. Tạo Virtual Environment

python -m venv venv
venv\Scripts\activate

3. Cài Đặt Dependencies

pip install -r requirements.txt

4. Chạy Server

python manage.py runserver
